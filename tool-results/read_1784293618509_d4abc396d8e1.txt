   741→    <div class="guide-section" id="algorithm">
   742→      <h2><span class="section-num">4</span> Trust Score Algorithm</h2>
   743→      <p>Each test receives a trust score from 0 to 100. The score is calculated from four weighted components plus an optional flaky penalty.</p>
   744→
   745→      <div class="algo-grid">
   746→        <div class="algo-card">
   747→          <div class="algo-weight">60% weight</div>
   748→          <h3>Pass Rate</h3>
   749→          <p>The percentage of recent runs that passed. A consistently passing test earns close to the full 60 points.</p>
   750→          <div class="algo-bar"><div class="algo-bar-fill" style="width:60%;background:var(--green)"></div></div>
   751→        </div>
   752→        <div class="algo-card">
   753→          <div class="algo-weight">20% weight</div>
   754→          <h3>Recent Trend</h3>
   755→          <p>Compares pass rate of the last 5 runs vs. the previous 5. Improving tests get a bonus; degrading tests lose points.</p>
   756→          <div class="algo-bar"><div class="algo-bar-fill" style="width:20%;background:var(--purple)"></div></div>
   757→        </div>
   758→        <div class="algo-card">
   759→          <div class="algo-weight">10% weight</div>
   760→          <h3>Duration Variance</h3>
   761→          <p>Measures coefficient of variation in execution time. Consistent durations suggest stability; erratic timing indicates flakiness.</p>
   762→          <div class="algo-bar"><div class="algo-bar-fill" style="width:10%;background:var(--yellow)"></div></div>
   763→        </div>
   764→        <div class="algo-card">
   765→          <div class="algo-weight">10% weight</div>
   766→          <h3>Error Pattern Consistency</h3>
   767→          <p>When a test fails, consistent error messages indicate a real bug, while varied errors suggest flakiness.</p>
   768→          <div class="algo-bar"><div class="algo-bar-fill" style="width:10%;background:var(--orange)"></div></div>
   769→        </div>
   770→        <div class="algo-card algo-penalty">
   771→          <div class="algo-weight" style="color:var(--red)">-5 to -20 penalty</div>
   772→          <h3 style="color:var(--red)">Flaky Category Penalty</h3>
   773→          <p>If a test is classified as flaky, an additional penalty is applied based on the category severity:</p>
   774→          <p style="margin-top:6px">
   775→            <code style="border-color:rgba(249,115,22,0.3);color:var(--orange)">timing</code> &minus;5 &nbsp;
   776→            <code style="border-color:rgba(139,92,246,0.3);color:var(--purple-light)">order_dependency</code> &minus;10 &nbsp;
   777→            <code style="border-color:rgba(239,68,68,0.3);color:var(--red)">shared_state</code> &minus;15 &nbsp;
   778→            <code style="border-color:rgba(234,179,8,0.3);color:var(--yellow)">non_deterministic_data</code> &minus;20
   779→          </p>
   780→        </div>
   781→      </div>
